VXML (EzXML)

VXML simplifies using of XML in Java, and provides reading/writing XML in few lines. VXML module contains following files:
  • EzXML.java

VINet (Violent Immediate mode Networking)

A thread-friendly, no-bloat, event-driven, and asynchronous TCP server for Violent Immediate mode Networking (Vinet) that is designed to be extremely easy and simple to use. The server manages client connections, communication, and various events such as client connection, disconnection, data reception, and exceptions.

VINet module contains following files:
  • VinetTransferableObjectUtility.java
  • TCP/VinetClientEventsTCP.java
  • TCP/VinetClientServer.java
  • TCP/VinetClientTCP.java
  • TCP/VinetServerClient.java
  • TCP/VinetServerEventsTCP.java
  • TCP/VinetServerTCP.java

Transferable Object Utility

Utility used to encrypt/decrypt data to send. 
public class VinetTransferableObjectUtility {
    public static <T> String assemble(T ...objects);
    public static String[] disassemble(String);
    public static boolean isAssembled(String);
}
  • public static <T> String assemble(T ...objects):

    This method assembles a string from the given objects, replaces single quotes, encodes it, and returns the result.

  • public static String[] disassemble(String):

    This method decodes an assembled string and splits it back into the original objects.

  • public static boolean isAssembled(String):

    This method checks if a string has been assembled by verifying its signature prefix.

TCP Client Events

Interface used as callback for client events. 
public interface VinetClientEventsTCP {
    void connection(VinetClientServer, VinetClientTCP);
    void disconnection(VinetClientServer, VinetClientTCP);
    void receive(String, VinetClientServer, VinetClientTCP);
    void event(String, String, VinetClientServer, VinetClientTCP);
    void exception(Exception, VinetClientTCP);
    void lost(VinetClientTCP);
    void timeout(VinetClientTCP);
}
  • void connection(VinetClientServer, VinetClientTCP):

    This method is triggered when the client connects to the server.

  • void disconnection(VinetClientServer, VinetClientTCP):

    This method is triggered when the client disconnects from the server.

  • void receive(String, VinetClientServer, VinetClientTCP):

    This method is triggered when the client receives data from the server.

  • void event(String, String, VinetClientServer, VinetClientTCP):

    This method is triggered when an event is toggled by the server, providing the event name and data.

  • void exception(Exception, VinetClientTCP):

    This method is triggered when an exception is raised on the client side in the `connect` method.

  • void lost(VinetClientTCP):

    This method is triggered when the client loses connection to the server.

  • void timeout(VinetClientTCP):

    This method is triggered when the connection times out.

    void exception(Exception, VinetServerTCP);
    void lost(VinetServerClient, VinetServerTCP);
}
  • void start(VinetServerTCP):

    Triggered when the server starts.

  • void end(VinetServerTCP):

    Triggered when the server stops.

  • void connection(VinetServerClient, VinetServerTCP):

    Triggered when a new client connects to the server.

  • void disconnection(VinetServerClient, VinetServerTCP):

    Triggered when a client disconnects from the server.

  • void receive(String, VinetServerClient, VinetServerTCP):

    Triggered when the server receives data from a client.

  • void event(String, String, VinetServerClient, VinetServerTCP):

    Triggered when a client triggers an event.

  • void exception(Exception, VinetServerTCP):

    Triggered when an exception is raised on the server side.

  • void lost(VinetServerClient, VinetServerTCP):

    Triggered when a client leaves the server without disconnecting properly.
